Q54. What is the output of this SQL query?

SELECT *
FROM employees
WHERE salary BETWEEN 50000 AND 70000
ORDER BY salary DESC;

A. Employees with salary between 50000 and 70000 in ascending order
B. Employees with salary between 50000 and 70000 in descending order
C. Employees with salary greater than 50000 and less than 70000 in descending order
D. All employees in the table
Answer: B. Employees with salary between 50000 and 70000 in descending order
Explanation: BETWEEN operator includes both endpoints. ORDER BY salary DESC sorts in descending order.

Q55. Which of the following SQL queries correctly retrieves the department name and the number of employees in each department?

Explanation: GROUP BY clause is used to group rows that have the same values in specified columns. COUNT(*) counts the number of rows in each group.
